# Fortran-Sandbox
*Testing environment for Fortran language.*

## Objectives
* Fortran
* OpenMP (multi-threading)
* gfortran, ifx, nvfortran
* fpm
* docker 
* jupyter
* linter

## Setup
---

### Standard installation
On Ubuntu/Debian

```
sudo apt update && sudo apt install gfortran -y
```


### Conda build

```
conda create -n fpm gcc gfortran 
```
